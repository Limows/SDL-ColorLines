# SDL-ColorLines

SDL Haiku and BeOS version

## Install Tools & Build

Haiku Recipe:

```sh
pkgman install libsdl*1.2-dev

cd ~/Projects/
git clone https://github.com/Limows/SDL-ColorLines
cd SDL-ColorLines/X86-Linux/
make clean
make
```

## Create package

## Controls

 - Cursor keys - directions
 - Enter - select
 - F6 - new game
 - F2 - pause music
 - ESC - back to menu
 - F4 - about

## Features

 - Game save your score automatically
 - Music
 - Animations
 - Mouse controls
